Implementation classes of the AST for CircusPattern.

This package provides implementations of the interfaces in {@link net.sourceforge.czt.circuspatt.ast}. As an application programmer, you should ONLY use the object factory provided in this package explicitly. Use the interfaces in {@link net.sourceforge.czt.circuspatt.ast} and the {@link net.sourceforge.czt.circuspatt.impl.CircusPatternFactoryImpl} when using the AST for CircusPattern.
