Implementation classes of the AST for CircusConf.

This package provides implementations of the interfaces in {@link net.sourceforge.czt.circusconf.ast}. As an application programmer, you should ONLY use the object factory provided in this package explicitly. Use the interfaces in {@link net.sourceforge.czt.circusconf.ast} and the {@link net.sourceforge.czt.circusconf.impl.CircusConfFactoryImpl} when using the AST for CircusConf.
