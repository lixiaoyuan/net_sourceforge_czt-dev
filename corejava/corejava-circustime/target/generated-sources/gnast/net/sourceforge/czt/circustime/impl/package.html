Implementation classes of the AST for CircusTime.

This package provides implementations of the interfaces in {@link net.sourceforge.czt.circustime.ast}. As an application programmer, you should ONLY use the object factory provided in this package explicitly. Use the interfaces in {@link net.sourceforge.czt.circustime.ast} and the {@link net.sourceforge.czt.circustime.impl.CircusTimeFactoryImpl} when using the AST for CircusTime.
